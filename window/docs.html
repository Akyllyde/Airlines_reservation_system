After installation, the user is prompted with a Sign-in / Sign-up modal. The sign-up tab can be used to create new user accounts. The Sign-in tab is used for logging in to the system by the Admin, Super User and User.

Booking a ticket has four steps in this system. Entering the parameters in the search form, then choosing one of the avilable tickets and providing credentials for payment and confirming the booking.

Ony the Admin can manage the fleet and its information, including adding, removing, editing aircraft information. The Fleet tab is used for fleet mangement.

The Admin and Super User have the permission to suspend, ban or delete user. But only the Admin has the authority to create new system user with role of Admin and Super User.

The Admin and Super User, They all have an unique dashboard. Admins dashboard shows revenue, orders, fleet info and so on. Whereas, the Super User's dashboard shows fleet info, user info, booking state and such.

The Admin and Super User can add, remove or modify booking data through the orders tab.

The software has plenty of Keyboard shortcuts avilable for ease of use. The shortcuts reference can be found in the 'Help' menu or by pressing Ctrl K

Feedback and suggestions or bug reporting is done through the about section of help menu.
